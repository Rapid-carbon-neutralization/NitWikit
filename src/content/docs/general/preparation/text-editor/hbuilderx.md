一个国产的文本编辑器。

:::caution

该软件存在一些问题，例如编译需要登录、命令行中夹带广告等（[来源](https://www.zhihu.com/question/47889106/answer/3189683653)）。

:::

### 关联右键菜单？

可以在工具 → 设置 (Ctrl + Alt + ，) 打开设置，找到“常用配置”手动选中【关联右键菜单】。
